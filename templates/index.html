Acne Type Detection System

Upload a clear, close-up image of your skin (JPEG format) for better results.

Note: This tool is for educational purposes only. Consult a dermatologist for professional advice.

How It Works

This tool uses a trained machine learning model to detect the type of acne from the uploaded image. It supports detection of three types: acne comedonica, acne conglobata, and acne papulopustulosa. The detected type is provided along with product recommendations.
